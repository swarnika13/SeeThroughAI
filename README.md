# SeeThroughAI – Real-Time Object Detection for Blind Users

## Overview
SeeThroughAI is an AI-powered assistive navigation system designed for visually impaired users. The project uses YOLO-based real-time object detection, computer vision, and voice-guided feedback to identify surrounding obstacles and provide intelligent positional awareness.

---

## Features
- Real-time object detection using YOLO
- Assistive AI-based navigation
- Voice-guided environmental feedback
- Left, right, and center obstacle positioning
- Real-time webcam inference
- Computer vision-based obstacle recognition

---

## Tech Stack
- Python
- OpenCV
- YOLOv5 / YOLOv8
- PyTorch
- Computer Vision
- Deep Learning
